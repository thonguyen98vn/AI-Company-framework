# Mobile App Domain Preset

This preset configures the AI Company Framework for building Mobile Applications.

## Included Standards
By activating this preset, the Execution Agents will default to Mobile-specific best practices:
- **Framework**: React Native with Expo (default) or Flutter.
- **State Management**: Zustand or Redux Toolkit.
- **Routing**: Expo Router.
- **UI/UX**: NativeWind (Tailwind for React Native) or default platform stylesheets.
- **App Stores**: Fastlane for automated deployments to App Store and Google Play.

## How to use
When the Founder Interview occurs, if the Founder mentions building a mobile app (iOS/Android), the system will automatically prioritize the rules from this folder to guide the engineering agents.
