# Getting Started

## 1. Installation

The easiest way to initialize a new AI Company Framework project is via the official NPM CLI:

```bash
npx create-ai-company my-project
```

The CLI will prompt you to select a template:
- **Minimal:** Only the core framework files.
- **Full:** Includes pre-configured Domain Presets (Mobile, Web3, GameDev, SaaS, Data Science).

## 2. Supported IDEs

Once initialized, open the `my-project` folder in your favorite AI IDE:
- **Cursor**
- **Windsurf**
- **GitHub Copilot**
- **Claude Code**
- **Gemini Advanced / Antigravity**

## 3. The First Message

In the AI Chat window of your IDE, simply type:

> "hello"

The AI will detect the `.cursorrules` (or equivalent file), realize the project is `UNINITIALIZED`, and automatically begin the **Founder Interview** to help you plan your architecture.
